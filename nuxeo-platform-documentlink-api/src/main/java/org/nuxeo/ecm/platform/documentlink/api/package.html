Provides API form the Application Level Proxy system.
The DocumentLinkAdapter object can be obtained using the standard adpater API :
   DocumentModel link = ...
   DocumentLinkAdapter adaptedLink = link.getAdapter(DocumentLinkAdapter.class);
This link adapter can be obtained on any document that implements the documentLink schema. The link adpater API offers methods to :
  • access the targeted link document
  • get/set properties on both the proxy and the linked document
